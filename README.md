# Multi-Robot Optimization

> Optimization-based Multi-Robot Task Scheduling and Performance Analysis

---

## 📌 Project Overview

This project investigates multi-robot collaborative task scheduling from the perspective of mathematical optimization.

Instead of focusing only on robotic system implementation, this project studies how to formulate multi-robot scheduling as a combinatorial optimization problem, solve it using optimization algorithms, and analyze the gap between practical solutions and theoretical optimal solutions.

---

## 🎯 Research Objectives

The project aims to answer three research questions:

1. How should multiple tasks be assigned to multiple robots?
2. How should collision-free paths be planned efficiently?
3. How close is the obtained solution to the theoretical optimum?
